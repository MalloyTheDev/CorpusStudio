# Dataset Versioning

Datasets should have explicit versions.

Example:

```text
coding_tutor_v0.1
coding_tutor_v0.2
coding_tutor_v1.0
```

## Track

- example count
- schema version
- export format
- quality report
- change notes
- created date
- source files

## Future

Support Git-backed dataset versioning, dataset cards, Evaluation report
comparison, and checkpoint comparison once local training launch is added.
