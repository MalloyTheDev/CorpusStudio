# Code of Conduct

Corpus Studio contributors are expected to keep discussion professional, constructive, and focused on building a useful dataset creation tool.

## Standards

Examples of positive behavior:

- giving specific technical feedback
- explaining tradeoffs clearly
- respecting project scope
- documenting decisions
- helping new contributors understand the system

Unacceptable behavior includes harassment, threats, abuse, or intentionally disruptive conduct.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, issues, and pull requests that violate these standards.
