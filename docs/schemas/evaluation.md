# Evaluation Schema

Used for model evaluation and regression testing.

## Example

```json
{
  "id": "math_001",
  "prompt": "What is 12 * 8?",
  "expected_answer": "96",
  "rubric": "Answer must be exactly 96.",
  "category": "arithmetic"
}
```

## Fields

- id
- prompt
- expected_answer
- rubric
- category
- difficulty
- tags
